# Spam Text/E-mail Classification using Machine Learning

This project implements a spam detection system using Natural Language Processing (NLP) and Machine Learning.

## Technologies Used
Python
Pandas
Scikit-learn
NLTK
TF-IDF

## Features
- Text preprocessing
- Stopword removal
- TF-IDF feature extraction
- Naive Bayes classifier
- Interactive spam prediction

## Example
Input: Congratulations! You won a free iPhone  
Output: Spam
